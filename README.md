# 🔐 KEY FORGE
### Secure Multi-Chain Wallet Generator

**Key Forge** — это мощный CLI-инструмент для массовой генерации криптовалютных кошельков.
Полностью оффлайн решение, модульная архитектура и шифрование военного уровня (AES-256-GCM).

## ⚡ Особенности

* **🛡 Безопасность:** Генерация происходит локально. Приватные ключи никогда не покидают ваше устройство.
* **🧠 Умное Ядро:** Система автоматически определяет требования модулей (seed, mnemonic, config) благодаря интроспекции.
* **📂 Шифрование:** Результаты сохраняются в `.enc` файлы, зашифрованные AES-256-GCM с использованием Scrypt KDF.
* **🧩 Модульность:** Легкое добавление новых блокчейнов через встроенный **Wizard** или плагины.
* **🎨 UI/UX:** Красивый интерфейс на базе `Rich` и `Questionary`.

## 🌍 Поддерживаемые Сети (из коробки)

| Сеть | Особенности |
|------|-------------|
| **Bitcoin (BTC)** | Поддержка **Taproot (BIP-86)**, Native Segwit, Legacy, Nested. |
| **Ethereum (EVM)** | ETH, BNB, Polygon, Avalanche и любые L2 сети. |
| **Polkadot (DOT)** | Поддержка **Sr25519** и работа через Mnemonic (совместимость с Talisman/Polkadot.js). |
| **Cosmos (ATOM)** | Генератор любых префиксов (Cosmos, Celestia, Osmosis, Dymension). |
| **Solana (SOL)** | Поддержка стандартов Phantom, Backpack, Solflare (`m/44'/501'/0'/0'`), Legacy-форматов и кастомных путей. |
| **SUI Network** | Генерация ED25519 ключей (Blake2b) для SUI. |

## 🚀 Установка и Запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/your-repo/key-forge.git
   cd key-forge
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv .venv
   # Windows:
   .venv\Scripts\activate
   # Mac/Linux:
   source .venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите:**
   ```bash
   python main.py
   ```
### ❗ Возможные ошибки при установке на Windows

Если во время выполнения команды `pip install` вы видите красный текст с ошибкой:
> `error: Microsoft Visual C++ 14.0 or greater is required. Get it with "Microsoft C++ Build Tools"`

Это значит, что вашей системе не хватает компилятора для сборки криптографических библиотек (`cryptography`, `substrate-interface`).

**Как исправить:**
1. Скачайте **Microsoft C++ Build Tools** с официального сайта Microsoft (или через Visual Studio Installer).
2. Запустите установщик.
3. В меню выбора компонентов поставьте галочку напротив **"Разработка классических приложений на C++"** (Desktop development with C++).
4. Нажмите "Установить".
5. После завершения установки снова запустите `pip install -r requirements.txt`.

## 🛠 Использование

### 1. Генерация кошельков
* Выберите `🚀 Сгенерировать кошельки` в меню.
* Выберите сеть (например, `Bitcoin`).
* Если сеть поддерживает настройки (как BTC, Cosmos или Polkadot), вам предложат выбрать формат адреса.
* Укажите количество кошельков и (опционально) Passphrase.
* Задайте пароль для шифрования итогового файла.

### 2. Расшифровка
* Выберите `🔓 Расшифровать файл`.
* Укажите путь к `.enc` файлу и пароль.
* Вы можете просмотреть ключи на экране или экспортировать их в **CSV**.

### 3. Добавление своих сетей (Wizard)
* Выберите `➕ Добавить сеть`.
* Мастер поможет вам создать новый модуль без написания кода (поддержка EVM, Cosmos SDK, Solana, Bitcoin-forks).

## 📦 Зависимости

Проект использует минимальный набор проверенных библиотек:
* `bip-utils` — основная криптография (BIP39/44/84/86).
* `PyNaCl` — безопасная и стабильная математика кривой Ed25519 (для Solana).
* `cryptography` — AES-GCM шифрование файлов.
* `substrate-interface` — работа с экосистемой Polkadot.
* `rich` & `questionary` — интерфейс.
