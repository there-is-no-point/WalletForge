# 🔐 WalletForge
### Secure Multi-Chain Wallet Generator

**WalletForge** — мощный CLI-инструмент для генерации, управления и защиты криптовалютных кошельков.
Полностью оффлайн решение, модульная архитектура и шифрование военного уровня (AES-256-GCM).

## 🆕 Обновление 2.1 (Синхронизация с GitHub)

### 🎨 Улучшенный Премиальный UI/UX
* **Живая конфигурация (Progressive Disclosure):** Интерактивная карточка параметров генератора динамически заполняется наверху экрана по мере ваших ответов, сохраняя консоль чистой. Длинные "простыни" вопросов в терминале остались в прошлом.
* **Навигация (Breadcrumbs):** Элегантные "хлебные крошки" (например, `🏠 Главное меню ➔ 🚀 Сгенерировать кошельки`) полностью заменили огромные полноэкранные ASCII-баннеры во всех подменю, освободив рабочее пространство.
* **Контекстные описания:** Главное меню и меню экспорта теперь снабжены стильными таблицами, детально описывающими каждую опцию до ее выбора.
* **Smart Clear:** Глубокая переработка алгоритмов очистки консоли (`prompt_toolkit`) для идеального отображения на всех версиях Windows без "артефактов" и наездов текста.

## ⚡ Особенности

* **🛡 Безопасность:** Генерация происходит локально. Приватные ключи никогда не покидают ваше устройство.
* **🧠 Умное Ядро:** Система автоматически определяет требования модулей (seed, mnemonic, config) через интроспекцию.
* **📂 Шифрование:** Результаты сохраняются в `.enc` файлы, зашифрованные AES-256-GCM с использованием Scrypt KDF.
* **🧩 Модульность:** Лёгкое добавление новых блокчейнов через плагины в директорию `networks`.
* **✨ Vanity-адреса:** Генерация красивых адресов с поддержкой мульти-паттернов, ETA и звуковым уведомлением.
* **🔪 Shamir Secret Sharing:** Разделение мнемоники на части (M-из-N) с шифрованным хранением.
* **🖨️ PDF-экспорт:** QR PDF и Paper Wallet PDF с линией сгиба для безопасного хранения.
* **✅ Верификация:** Проверка что мнемоника → адрес совпадают (без утечки ключей).
* **🎨 Премиальный UI/UX:**
  * **Живая конфигурация (Progressive Disclosure):** Интерактивная карточка параметров динамически заполняется наверху экрана по мере ваших ответов, сохраняя консоль чистой.
  * **Навигация:** Элегантные `Breadcrumbs` (Хлебные крошки) для отслеживания пути в подменю.
  * **Контекст:** Понятные Rich-таблицы с описанием каждой опции меню.

## 🚀 Установка и Запуск

1. **Клонируйте репозиторий:**
   ```bash
   git clone https://github.com/there-is-no-point/WalletForge.git
   cd WalletForge
   ```

2. **Создайте виртуальное окружение (рекомендуется):**
   ```bash
   python -m venv .venv
   # Windows:
   .venv\Scripts\activate
   # Mac/Linux:
   source .venv/bin/activate
   ```

3. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Запустите:**
   ```bash
   python main.py
   ```

> **Windows:** Можно использовать `start.bat` для автоматической активации venv и запуска.

### ❗ Ошибка при установке на Windows

Если видите ошибку:
> `error: Microsoft Visual C++ 14.0 or greater is required`

1. Скачайте **Microsoft C++ Build Tools** (Visual Studio Installer).
2. Установите компонент **"Разработка классических приложений на C++"**.
3. Повторите `pip install -r requirements.txt`.

## 🛠 Использование

### 🚀 Генерация кошельков
* Выберите сеть, количество кошельков, длину мнемоники и Passphrase.
* Результат сохраняется в зашифрованный `.enc` файл.

### 🔓 Расшифровка и экспорт
* Расшифруйте `.enc` файл паролем.
* Экспорт в: **QR PDF**, **Paper Wallet PDF**, **JSON** или просмотр на экране.

### ✨ Vanity-адреса
* Поиск адресов с заданным текстом (Префикс / Суффикс / Содержит).
* **Мульти-паттерн:** поиск нескольких вариантов одновременно (`dead,beef,cafe`).
* **ETA:** оценка оставшегося времени поиска.
* **Звуковой сигнал** при нахождении.
* Валидация символов под каждую сеть.

### 🧩 Shamir Secret Sharing
* Разбейте мнемонику на N частей с порогом M (например, 3-из-5).
* Сохранение: текстовые файлы (.txt) или зашифрованные (.enc).
* Восстановление: из файлов, зашифрованных файлов или ручной ввод.

### ✅ Верификация кошельков
* Проверка что мнемоника корректно воспроизводит адрес из файла.
* Поддержка всех форматов (BTC Taproot, Polkadot Sr25519, и т.д.).

### 🧩 Добавление своих сетей
* Поддержка создания кастомных плагинов сетей в папке `networks/`.
